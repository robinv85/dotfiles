Installation
------------

Clone this repo anywhere, e.g. to ~/dots, then run bootstrap.sh and the following files & folders are automatically symlinked into your $HOME directory.

    .profile
    .bash_profile
    .bashrc
    .vim/
    .vimrc
    .osx
